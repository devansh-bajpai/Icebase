## Introduction
Icebase is a full-stack dashboard application that allows users to manage API keys, track logs, and visualize data using Chart.js. It integrates authentication, API services, and a real-time logging system, providing a scalable and user-friendly interface for web applications.

## Demo Video Link
<a href="#">https://youtu.be/UrrttZF2kMQ</a>

## Features
- User authentication and session management  
- Create, revoke, and copy API keys  
- Track and display logs per API key  
- Interactive charts and data visualization using Chart.js  
- Real-time updates via sockets  
- Responsive dashboard layout  

### Facial Verification Features
- **Uses Faster Model** as well as higher accuracy than most available  
- Does the facial data search in **logN time complexity** rather than N  
- **Future:** Liveness check feature using customized movements or actions  
- Can be used for multiple cases: authentication system, attendance, captcha, etc.  
- **Plug & Play JavaScript SDK** – Add face verification to any web app with just a few lines of code  
- **Developer Friendly** – Completely customizable, reusable, and easy integration  
- **End-to-End Encryption** – All face data and responses are securely encrypted and stored  
- **Cloud-Processed** – No need for GPUs or heavy ML libraries on your server  

## Technology Stack
1. MERN (MongoDB, Express, React, Node.js)  
2. JWT for authentication  
3. Chart.js for data visualization  
4. Socket.io for real-time updates  
5. Axios for API calls
